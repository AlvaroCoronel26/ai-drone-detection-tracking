# AI-Based Drone Detection and Tracking System

Engineering thesis project focused on real-time drone detection and tracking using artificial intelligence and computer vision techniques.

## Overview

This project implements a drone detection and tracking architecture using YOLOv8 neural networks integrated with motorized tracking systems.

The system was developed as part of a Mechatronics Engineering thesis project at UPC (Universidad Peruana de Ciencias Aplicadas).

## Main Features

- Real-time drone detection using YOLOv8
- Object tracking system
- PID-based target positioning control
- Raspberry Pi integration
- Communication between processing systems and embedded hardware
- Computer vision architecture using Python and OpenCV

## Technologies Used

- Python
- YOLOv8
- OpenCV
- Raspberry Pi
- PID Control
- Computer Vision

## Repository Goals

This repository is intended to document the development process, experiments, and implementation of intelligent drone tracking systems using AI vision technologies.
